# Compilers



### How to use

| Compilers | Platform               | Command                                                          |
|-----------|------------------------|------------------------------------------------------------------|
| GCC       | Linux, among others... | g++ -std=c++0x example.cpp -o example_program                    |
| CLANG     | OS X, among others...  | clang++ -std=c++11 -stdlib=libc++ example.cpp -o example_program |
